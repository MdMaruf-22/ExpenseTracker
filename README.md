# Expense Tracker

## Overview

The Expense Tracker is a project designed to help users manage their personal finances by tracking income and expenses. It provides an intuitive interface to add, edit, and view financial transactions, along with insightful summaries and reports.

## Features

- Add and categorize expenses and income.
- View transaction history.
- Generate financial summaries and reports.
- User-friendly interface.

## Technology Used

- **Java**: The core programming language used for this project.
- **Java Swing**: Used for building the graphical user interface (GUI).

## Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```bash
   cd EXPENSE
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

## Usage

1. Start the development server:
   ```bash
   npm start
   ```
2. Open your browser and navigate to:
   ```
   http://localhost:3000
   ```
